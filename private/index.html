Controles Direccionales

  • Flecha Arriba (W): Mover el robot hacia adelante
  • Flecha Izquierda (A): Girar el robot a la izquierda
  • Flecha Derecha (D): Girar el robot a la derecha
  • Flecha Abajo (S): Mover el robot hacia atrás
  • Botón Central: Activar el claxon del robot

Control de Velocidad

  • Lento (V): 33% de velocidad - Ideal para espacios reducidos
  • Medio (B): 66% de velocidad - Velocidad estándar
  • Rápido (N): 100% de velocidad - Máxima potencia

Vista de Cámara

  • Transmisión en tiempo real desde la cámara del robot
  • Modo Automático: El robot navegará de forma autónoma
  • Pantalla Completa: Para una mejor visualización

Consejos

  • Usa velocidades bajas en espacios con obstáculos
  • El registro del sistema muestra el estado y eventos importantes
